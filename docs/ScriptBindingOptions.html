bool SplitTypeListIntoHookupFiles false Whether or not to create hookup files (see documentation on hookup files for more info). Hookup files contain generated code that binds reflected classes to scripting languages using the same macros as the type list. If this is true, one hookup file (*.hookup.cpp) will be created for each reflected header file.
